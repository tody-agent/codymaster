# Pre-Publish Checklist — CM Video Factory

Run through this checklist before ANY video upload. The Policy Compliance Engine automates most checks, but human review is required for items marked 👤.

---

## Phase 1: Content Review

- [ ] **Script Quality** — Does the script provide genuine value (educational, entertainment)?
- [ ] **Hook Effectiveness** — Does the first 3 seconds create curiosity? No generic "Hey guys" intros?
- [ ] **CTA Clarity** — Is the call-to-action specific and actionable?
- [ ] 👤 **Human Review** — Has a human reviewed and approved the script content?
- [ ] **No Banned Content** — Script scanned for hate speech, misleading claims, harmful content?
- [ ] **No Unauthorized Cloning** — No real person's voice/face used without consent?

## Phase 2: Technical Quality

- [ ] **Audio Quality** — TTS audio is clear, properly paced, no artifacts?
- [ ] **Subtitle Sync** — Subtitles match spoken words with <200ms tolerance?
- [ ] **Visual Quality** — 1080p minimum, no rendering artifacts, smooth animations?
- [ ] **Duration Check** — Within platform guidelines? (TikTok: 15-180s, YouTube: 60-900s)
- [ ] **Safe Zones** — No text/UI in platform-obscured areas (TikTok: bottom 200px, top 150px)
- [ ] **Audio Levels** — Normalized to -14 LUFS for consistent volume?

## Phase 3: Policy Compliance

- [ ] **AI Disclosure Label** — Metadata includes `ai_generated: true`?
- [ ] **AI Disclosure Text** — Video description includes AI disclosure statement?
- [ ] **Visual Uniqueness** — ≥70% visual difference from last 5 uploaded videos?
- [ ] **Hashtag Count** — 3-7 hashtags (TikTok), 5-15 tags (YouTube)?
- [ ] **No Banned Hashtags** — None of: #fyp, #foryou, #viral, #trending?
- [ ] **Upload Rate** — Within daily limit? (TikTok: ≤3/day, YouTube: ≤2/day)
- [ ] **Cooldown Period** — Sufficient time since last upload? (TikTok: 4h, YouTube: 6h)

## Phase 4: Platform-Specific

### TikTok
- [ ] Format: 9:16 (1080×1920)?
- [ ] Duration: 15-60s (sweet spot) or up to 180s?
- [ ] TikTok AI label tool used during upload?
- [ ] Caption text not blocked by comment/like buttons?

### YouTube Shorts
- [ ] Format: 9:16 (1080×1920)?
- [ ] Duration: ≤60s?
- [ ] #Shorts tag included?
- [ ] Mobile-friendly text sizes?

### YouTube Long-form
- [ ] Format: 16:9 (1920×1080)?
- [ ] Duration: 3-15 minutes?
- [ ] Custom thumbnail uploaded?
- [ ] SRT subtitle file uploaded?
- [ ] Creator Studio AI disclosure toggle activated?
- [ ] 👤 Unique description written (not copy-paste)?
- [ ] End screen/cards configured?

### Instagram Reels
- [ ] Format: 9:16 (1080×1920)?
- [ ] Duration: 15-90s?
- [ ] Cover image selected?

## Phase 5: Final Verification

- [ ] 👤 **Watch Full Video** — Human watches the rendered video end-to-end
- [ ] **File Size** — Under platform limit? (TikTok: 287MB, YouTube: 256GB)
- [ ] **Filename** — Descriptive name (not `output_001.mp4`)?
- [ ] **Publish Status** — Set to Private/Draft first for final review?
- [ ] **Schedule** — Optimal posting time for target audience?
