How it Works

1

Install the Toolkit

Copy the Doc-Kit folder into your Antigravity skills repository. cp -r doc-kit ~/.gemini/antigravity/skills/

2

Generate Prompt

Run the interactive CLI script to pick your documentation type (API, SOP, Tech) and output format (Markdown or Docusaurus).

3

Let Antigravity Work

Paste the generated prompt into your Google Antigravity session. The agent orchestrates 4 sub-skills to analyze, map, and write.

4

Deploy

You now have a folder of structured markdowns, or a fully functional Docusaurus site ready to deploy to Vercel/Netlify.
