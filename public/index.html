UNDER THE HOOD

The RARV Execution Cycle + 6-Gate Quality

Every task assigned to Cody Master goes through an autonomous loop ensuring production-ready code.

1. Read (Context)

Agent reads instructions, design specs, APIs, and project guidelines to build Working Memory.

2. Act (Execute)

Agent writes tests FIRST, then writes the implementation code ensuring zero regressions.

3. Review (Gate)

Runs tests, linter, type checks, and security scans. If it fails, auto-loops back to Act.

4. Verify (Human)

Agent presents a walkthrough and UI screenshot for your final human approval.
