The investigation
🚨 Bug Report
"Login button doesn't respond on iPhone"

🔎 CodyMaster found the problem:
The code used an event that iPhones
don't understand ('mousedown').

🛠️ The fix:
Switched to a universal event that
works on ALL devices.
